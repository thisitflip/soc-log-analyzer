# SOC Log Analyzer

Projeto simples para detectar atividades suspeitas em logs.

## Objetivo

Identificar possíveis ataques de brute force analisando logs de autenticação.

## Funcionalidades

- Detecta múltiplas tentativas de login falhadas
- Identifica IPs suspeitos
- Analisa logs simples de autenticação

## Tecnologias

Python 3

## Como usar

Execute o script:

python analyzer.py
